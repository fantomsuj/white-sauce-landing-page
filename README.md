# White Sauce Grill & Bar - Landing Page

Landing page for White Sauce Grill & Bar, a Halal Mediterranean restaurant in Downtown Chapel Hill, NC.

**Live Site**: https://fantomsuj.github.io/white-sauce-landing-page/

## About

White Sauce Grill & Bar serves chicken, gyro, and falafel over rice, pita, or fries with an array of housemade sauces. Located at 104 W Franklin St, Chapel Hill, NC.

## Features

- Single-page responsive design
- Mobile-first layout
- Scroll-triggered animations
- Interactive menu with category tabs
- Sauce showcase section
- Customer reviews
- Location with hours and contact info

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript (Intersection Observer for animations)
- Google Fonts (Fraunces, DM Sans)
